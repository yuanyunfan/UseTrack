## Architecture Notes

- Swift entry point 用 `UseTrackCollector.swift` 而非 `main.swift`（`@main` 与 top-level code 冲突）
- Swift 和 Python 通过 SQLite 文件共享数据（WAL 模式支持并发读写）
- Python MCP Server 使用 FastMCP v2+ 的 `@mcp.tool()` 装饰器
- 产品设计文档: Obsidian `00-Inbox/20260410-Mac个人效率监控系统产品设计.md`
